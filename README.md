# 📊 Job Skill Demand Forecasting

A full-stack AI/ML project that analyzes and forecasts in-demand job skills using machine learning and data-driven insights.

## 🚀 Features

- Skill demand forecasting
- Data analysis and visualization
- Machine learning prediction models
- FastAPI backend
- React frontend interface
- Interactive dashboards

## 🛠️ Tech Stack

### Backend
- Python
- FastAPI
- Pandas
- Scikit-learn

### Frontend
- React.js
- Axios
- CSS
